Add Text to PDF

Place text anywhere on a PDF, drag to position, set size and color, then download. Runs in your browser; your file is never uploaded.

Upload a PDF to start adding text.
Added text is permanently merged into the PDF and saved as a new file. All languages (incl. Korean) are supported. (It overlays new text; it does not edit the PDF's existing text.)

How to add text to a PDF

1) Upload a PDF → 2) click 'Add text', type & drag → 3) download

Everything runs in your browser, so your file is never uploaded.

Can it edit the PDF's existing text?

No. It overlays new text; editing the original embedded text is not supported.

Does Korean work?

Yes — all languages and emoji are supported.
