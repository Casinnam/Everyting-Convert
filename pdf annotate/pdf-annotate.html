Add Text to PDF

Place text anywhere on a PDF, drag to position, set size and color, then download. Runs in your browser; your file is never uploaded.

1 / 1
Upload a PDF to start adding text.
Adding new text is free. To edit the PDF's existing text (erase & replace), turn on ‘Edit text’ and click any text — free with Pro, otherwise 3 credits per download. It works best on solid-background (e.g. white) documents. The result is merged into a new PDF and supports all languages (incl. Korean).

How to add text to a PDF

1) Upload a PDF → 2) click 'Add text', type & drag → 3) download

Everything runs in your browser, so your file is never uploaded.

Can it edit the PDF's existing text?

Yes. Turn on ‘Edit text’ and click any text to erase that spot and retype it. This is free for Pro members, otherwise 3 credits per download. It covers the original with the background color and draws the new text in the same spot, so it works best on solid-background (e.g. white) documents.

Does Korean work?

Yes — all languages and emoji are supported.
