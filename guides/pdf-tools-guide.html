PDF Guide

How to choose the right PDF tool

PDF files can contain text, images, scanned pages, forms, tables, and complex layouts. Choosing the right tool helps you avoid poor formatting and repeated work.

PDF to Word

Use this when you need editable text. It works best with text-based PDFs. Scanned PDFs may require OCR, and complex layouts can need manual cleanup.

PDF to Excel

Use this for tables, reports, invoices, and lists. Enhanced table detection helps with structured data, but very messy scans may still need review.

PDF to JPG

Use this when you need each page as an image for previews, uploads, thumbnails, or sharing.

Merge, split, rotate, and remove pages

Use PDF tools when the file is already in the right format but the page order, size, or page selection needs adjustment.

Before uploading a PDF

  • Check whether the PDF opens correctly.
  • Use a clear copy for scanned files.
  • For private documents, remove unnecessary pages before conversion.
  • After conversion, check page order, tables, and special characters.

PDF 가이드

상황에 맞는 PDF 도구 선택하기

PDF에는 텍스트, 이미지, 스캔 페이지, 양식, 표, 복잡한 레이아웃이 함께 들어갈 수 있습니다. 목적에 맞는 도구를 고르면 형식 깨짐과 반복 작업을 줄일 수 있습니다.

PDF를 Word로 변환

문장을 수정해야 할 때 사용하세요. 텍스트 기반 PDF에서 가장 잘 작동합니다. 스캔 PDF는 OCR이 필요할 수 있고, 복잡한 레이아웃은 변환 후 손질이 필요할 수 있습니다.

PDF를 Excel로 변환

표, 보고서, 청구서, 목록 데이터를 꺼낼 때 사용하세요. 표 감지 강화 기능은 구조화된 데이터에 도움이 되지만, 흐릿한 스캔본은 확인이 필요합니다.

PDF를 JPG로 변환

각 페이지를 이미지로 만들어 미리보기, 업로드, 썸네일, 공유에 사용하고 싶을 때 적합합니다.

병합, 분할, 회전, 페이지 제거

파일 형식은 이미 PDF로 맞지만 페이지 순서, 필요한 페이지, 방향, 용량을 조정해야 할 때 사용하세요.

PDF를 업로드하기 전에

  • PDF가 내 기기에서 정상적으로 열리는지 확인하세요.
  • 스캔 파일은 글자가 선명한 복사본을 사용하세요.
  • 민감한 문서는 변환 전에 불필요한 페이지를 제거하세요.
  • 변환 후 페이지 순서, 표, 특수문자를 꼭 확인하세요.
