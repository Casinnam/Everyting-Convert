PDF & Documents

Lock and unlock your PDF with a password

A password is the simplest way to protect a sensitive PDF — a contract, a payslip, an ID scan — before you share it. This tool applies 256-bit AES encryption so the document asks for a password when opened, and it can also remove protection again using a password you already know. Everything runs in your browser through WebAssembly, so neither the file nor the password ever leaves your device.

Best results

  • Use a long, unique password that is hard to guess
  • Store the password somewhere safe and separate
  • Keep the original file before you protect it

Common fixes

  • Unlock failed? Check the password and its capitalization
  • Already protected? Unlock first, then protect with a new password
  • Forgot the password? It cannot be recovered, by design

Private by design

Encryption and decryption run entirely in your browser using client-side WebAssembly. Your PDF and password are never uploaded, stored, or shared, so confidential files stay on your device.

Frequently asked questions

Is the PDF password tool free?

Yes. You can protect and unlock PDFs for free for everyday use, and Pro removes daily limits.

Are my files uploaded to a server?

No. Encryption and decryption run in your browser using WebAssembly, so the PDF and your password never leave your device.

What encryption is used?

Protected PDFs use 256-bit AES encryption, the strong standard supported by Adobe Acrobat and other modern readers.

Can you unlock a PDF if I forgot the password?

No. Unlocking only removes a password you already know. We cannot crack or recover forgotten passwords, which is what keeps encryption secure.

Will the protected PDF open everywhere?

Yes. The output is a standard encrypted PDF that prompts for the password in Acrobat, Preview, Chrome, and other readers.
